# Hammadh Arquil

Welcome to my personal portfolio. This site serves as a digital garden for my projects, writing, and professional journey, built with a focus on performance and simplicity.

## Built With

*   **Astro**: For top-tier performance and content delivery.
*   **React**: For interactive UI components.
*   **Tailwind CSS**: For custom design and typography.
*   **Nanostores**: For lightweight state management.

## Features

*   **Dark Mode**: A thoughtful theme toggle for comfortable reading.
*   **Blog**: A markdown-based content collection for sharing knowledge.
*   **Responsive**: Optimized for devices of all sizes.
*   **Performance**: Static site generation for near-instant load times.

## Development

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start the server**:
    ```bash
    npm run dev
    ```

## Structure

```
/src
  /components    # Navigation, ThemeToggle, interactive islands
  /content       # Blog posts (Markdown)
  /pages         # Routes (Home, About, Work, Blog)
  /styles        # Global styles & Tailwind config
```
